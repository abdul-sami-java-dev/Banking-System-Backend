# Banking System API

A secure and scalable Banking System backend built with Spring Boot, designed to simulate core banking operations through RESTful APIs. The application provides account management, fund transfers, transaction tracking, authentication, authorization, and administrative capabilities while following clean architecture and industry-standard backend development practices.

## Features

### Authentication & Authorization

* User Registration and Login
* JWT-based Authentication
* Spring Security Integration
* Role-Based Access Control (Admin/User)

### Account Management

* Create Bank Accounts
* View Account Details
* Check Account Balance
* Account Status Management
* Support for Multiple Account Types

### Transaction Management

* Deposit Funds
* Withdraw Funds
* Transfer Funds Between Accounts
* Automatic Transaction Recording
* View Transaction History

### Security

* JWT Token Validation
* Protected API Endpoints
* Password Encryption using BCrypt
* Secure Access Control

### API Documentation

* Swagger/OpenAPI Integration
* Interactive API Testing Interface

## Architecture

The project follows a layered architecture:

* Controller Layer
* Service Layer
* Repository Layer
* Security Layer
* Database Layer

This separation of concerns improves maintainability, scalability, and testability.
